# ПЛАН 2 — Лёгкий локальный переводчик для слабых ПК

## Цель

Найти и собрать оптимальный офлайн-переводчик для ПК без GPU с ≤16 GB RAM:
- переводит слова и длинные тексты RU↔EN (и другие пары)
- работает быстро и без интернета
- прост в установке и использовании

**Приоритет:** сначала тестируем **готовые решения** (не нужно ничего собирать), затем **лёгкие модели** с минимальной обвязкой.

---

## Целевой профиль железа

| Параметр | Минимум | Комфорт |
|----------|---------|---------|
| CPU | любой x86-64, 2+ ядра | 4+ ядра |
| RAM | 8 GB | 16 GB |
| GPU | отсутствует | отсутствует |
| Диск | SSD желателен | SSD |
| ОС | Windows 10 / Linux | любая |

---

## Часть I — Готовые решения (тестируем в первую очередь)

### Уровень 1 — Установил и запустил (нет кода, нет настройки)

| # | Решение | Суть | Качество (ожидаемое) | Вес |
|---|---------|------|---------------------|-----|
| R1 | **Argos Translate** (GUI + CLI + Python) | OpenNMT-based, офлайн, пакеты моделей | умеренное | ~300 MB |
| R2 | **translateLocally (Bergamot)** | Marian-based desktop app, офлайн | умеренное | ~100 MB + модель |
| R3 | **Firefox Translations** (расширение) | WASM + Bergamot, офлайн в браузере | умеренное | встроено |

### Уровень 2 — Установка через pip/docker, есть UI

| # | Решение | Суть | Качество (ожидаемое) | Вес |
|---|---------|------|---------------------|-----|
| R4 | **LibreTranslate** (self-hosted) | Argos backend + REST API + web UI | умеренное | ~500 MB + модели |
| R5 | **DesktopTranslator + M2M-100 1.2B** | CTranslate2 GUI, M2M100 | хорошее | ~5 GB |
| R6 | **LTEngine** (LibreTranslate-compatible, llama.cpp) | LLM backend + API, полностью офлайн | среднее | зависит от модели |

### Уровень 3 — Требует Ollama / llama.cpp, но UI есть

| # | Решение | Суть | Качество (ожидаемое) | Вес |
|---|---------|------|---------------------|-----|
| R7 | **Local-Translator + Ollama + TranslateGemma 4B** | HTML UI + PDF/DOCX extraction + Ollama | отличное | ~8 GB |
| R8 | **TranslateGemma-Studio + llama.cpp + GGUF** | Desktop UI, chunking, офлайн | отличное | ~4–8 GB |

## Критерии оценки готовых решений

| Критерий | Вес | Как измеряем |
|----------|-----|-------------|
| Качество перевода RU↔EN | ★★★ | chrF + субъективная оценка (1–5) |
| Скорость на CPU | ★★★ | сек/абзац на тестовом железе |
| RAM в пике | ★★★ | `Task Manager` / `psutil` |
| Лёгкость установки | ★★ | шаги до первого перевода |
| Поддержка длинных текстов | ★★ | тест документа 5000+ слов |
| Сохранение форматирования | ★ | тест DOCX / PDF round-trip |
| Офлайн-режим | ★★★ | тест без интернета |
| Поддержка других языков | ★ | тест EN↔DE |

## Тестовый корпус (упрощённый, без GPU)

### Блок A — Слова (быстрый тест)
- 50 слов: бытовые (20) + технические (20) + сложные/сленг (10)
- EN→RU и RU→EN

### Блок B — Предложения
- 30 предложений разной сложности и стиля
- EN→RU и RU→EN

### Блок C — Абзацы (основной тест)
- 10 абзацев ~200 слов: технические (5) + научные (3) + художественные (2)
- EN→RU и RU→EN

### Блок D — Длинный документ
- 1 документ ~3000–5000 слов (научная статья или техническое руководство)
- Форматы: plain text + DOCX (для решений с document support)
- EN→RU
